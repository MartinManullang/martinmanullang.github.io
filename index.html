Project

Indirect Communication

This application is a medium of learning between educators and students. the concept of this design is a combination of zoom and classroom designs where this application can carry out online learning like zoom meetings and can also view lists of material, assignments and can collect assignments. So the design of this application can later help teachers such as teachers or lecturers in providing material and assignments as well as being able to carry out online discussions which can help communication between teachers and students

Project

Game Top Up

This application is an application that helps players if they want to top up the game. Some games require players to make purchases if they want to get something provided by the game, for example in the legendary mobile game. In the game there are diamonds that can be exchanged for several skins, heroes, emblems and so on. To get these diamonds, you have to purchase them with real money and in the design of this application it helps players top up the game

Project

Mental health services

This application is an Android-based mental health application. This application helps people who want to do counseling about their mental health with several doctors who know the field. In this application there are several features such as screening, articles, chat with bots and also a list of existing doctors. The design of this application is easy to use and is expected to help people who wish to counsel their mental health problems

With the design of this application, it is hoped that it can help someone who wants to find shoes quickly and easily.
